when you hire a freelancer to create your website the usual deliverables you'll get is the HTML, CSS and Javascript code together with all the visual assets.

working with me you'll get more:

a living styleguide

the User Interface Styleguide is a visual documentation tool presenting all the components used to build the final website.

with a styleguide anyone in the future can pick your project and move forward; no vendor lock-in.

atomic web design

modern websites are built on reusable components instead of monolithic blocks.

with granular components you can easily fine-tune / change / adapt your website design to any future requirements.

mobile first design

progressive enhancement or grateful degradation? mobile first or desktop first?

to create perfect experience on all devices the best practice today is to build from bottom-up

project timeline

the project timeline is a documentation tool. it marks milestones together with business and design decisions.

it is the short history of what was build, and why that particular way.

standards compliance

semantic HTML5, valid HTML5, modern browser support, usability and web development best practices

integration with ecosystems

analytics, social networks, marketing platforms, performance monitoring
