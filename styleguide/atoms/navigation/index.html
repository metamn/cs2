List inline

@mixin list-inline($margin: $pad) {
  li {
    display: inline-block;
    float: left;
    margin-right: $margin;
    
    &:last-of-type {
      margin-right: 0;
    }
  }
}
A list which displays it's elements from left to right, not below each other.
