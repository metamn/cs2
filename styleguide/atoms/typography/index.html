Typography

Font size, line height and pad

The default font size and line-height. This sets the site's basic vertical rhythm. For more please consult http://lamb.cc/typograph/

$font-size:100% sets the basic font size and the em to 16px; $line-height: 1.25 sets the height of a line to 20px, or 1.25em.

The default em value will later be used for spacing elements on the site. This way spacing of elements will stay in sync with the typography making an aesthetically pleasant grid.

  • $font-size: 100%;

  • $line-height: 1.25;

  • $pad: 1.25em;

This text is positioned to 1 $pad below the list above.

Heading size and padding

  • $heading-size: 1.75em;
  • $heading-padding: 1.51785em;

In modern typography headings don't differ by size but decoration. All headings will have the same size preserving the vertical rhythm by using the Traditional em sizes from Lamb's Typography Calculator.
